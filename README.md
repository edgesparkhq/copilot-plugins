# EdgeSpark Copilot Plugins

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)

Official Copilot plugins by EdgeSpark.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [edgespark](./plugins/edgespark) | Build full-stack apps on EdgeSpark with database, auth, storage, and deployment workflows |

## Installation

### 1. Add the marketplace (one-time)

```bash
copilot plugin marketplace add edgesparkhq/copilot-plugins
```

### 2. Install a plugin

```bash
copilot plugin install edgespark@edgespark-copilot-plugins
```
